# Mitos y Relatos Santiagueños

Colección de mitos, leyendas y relatos tradicionales de Santiago del Estero, Argentina.

## Estructura del Proyecto

```
mitos-santiagueños/
├── src/
│   ├── content/
│   │   └── mitos/          # Archivos MDX de cada mito
│   ├── pages/
│   │   ├── index.astro     # Página principal con lista
│   │   └── mitos/
│   │       └── [slug].astro # Página individual de cada mito
│   └── layouts/
│       └── Layout.astro
├── public/
│   └── images/             # Imágenes de los mitos
└── package.json
```

## Mitos Incluidos

1. **La Telesita** - La bailarina que murió de sed y fuego
2. **El Kakuy** - El hombre pájaro que vive en el monte
3. **El Crespín** - El duende de la suerte
4. **La Dama de Blanco** - El alma en pena del centro
5. **El Nahuelito Santiagueño** - La criatura del río Dulce
6. **El Tunal Milenario** - Los cactus sagrados con poder curativo
7. **El Duende Tragaldabas** - El comelón nocturno
8. **La Luna del Diablo** - La luna roja de mal augurio
9. **Los Siete Lagos** - Las lagunas misteriosas
10. **El Lobizón** - La bestia de la noche
11. **El Pombero** - El guardián del monte
12. **La Novia del Diablo** - La tragedia de Dolores

## Tecnologías

- Astro
- Tailwind CSS
- MDX
