## INSTRUCCIONES PARA GEMINI/IMAGEN 3

1. **Aspect ratio:** 16:9 horizontal para todas las imágenes
2. **Style:** Usar el estilo base al inicio de cada prompt
3. **Consistency:** Mantener la paleta de colores tierra (ocres, marrones, rojos terracota, verdes oliva) en todas las imágenes
4. **Technique:** Ilustración tipo acuarela y tinta sobre papel
5. **Atmosphere:** Mística, tradicional, de libro de mitología latinoamericana

**Nota:** Si Gemini permite, generar primero una imagen de referencia con el estilo base y luego usar esa como referencia para las demás para mantener consistencia.
